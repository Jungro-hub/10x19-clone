# web-clone

    프론트엔드 개발자를 준비하며 SCSS, JS를 사용하여 웹사이트를 클론코딩하였습니다.

## criteria

    실험적이면서도 모던한 디자인적 요소가 있으며 인터랙티브한 사이트들을 선정하였습니다.
